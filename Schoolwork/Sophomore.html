FPGA - Morse Code Decoder

Basys 3 FPGA | Verilog | Xilinx Vivado | Team of 2

Developed a fully functional Morse code decoder on a Basys 3 FPGA using Verilog in Xilinx Vivado. The project allowed users to input Morse code via a single push button, which was then decoded and displayed on the on-board seven-segment display.

  • Designed and implemented multiple Finite State Machines (FSMs) to handle timing logic for dot, dash, character spacing, and word spacing detection.
  • Created a clock divider to manage signal timing for interpreting user input accurately.
  • Built a Morse code tree traversal FSM to decode sequences into the correct alphanumeric characters.
  • Used Karnaugh maps (K-maps) to derive simplified Boolean expressions for display control logic.
  • Developed a comprehensive testing strategy, validating each Morse character individually and in sequence.
  • We tried to get the letters to output in a scrolling pattern but were unable to before we ran out of time.

Autonomous Obstacle-Avoidance Robot

Zumo 32U4 OLED Robot | Arduino C++ | Real-Time Sensor Navigation

Programmed a Zumo 32U4 robot to autonomously detect and avoid objects using ultrasonic sensors. The robot constantly scans its environment and adjusts motion in real time based on measured distance from obstacles.

  • Interfaced ultrasonic sensor input with the Zumo to detect proximity of obstacles in front of the robot.
  • Developed an embedded C++ algorithm to analyze sensor data continuously and modify motor speed/direction on the fly.
  • Displayed simple messages on the OLED screen, such as direction changes or object proximity.
  • Tuned the robot's movement to avoid obstacles smoothly and operate reliably in different environments.
